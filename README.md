# Marketing Funnel & Conversion Performance Dashboard |

An interactive data-driven dashboard built using Streamlit to analyze marketing campaign performance, identify user drop-offs, and provide actionable business insights.

## Dataset:

Nykaa Marketing Campaign Performance Dataset; Real-World Inspired E-commerce Campaign Data for Marketing Analytics & ROI.

## Live Dashboard:
https://nykaa-campaign-dashboard.streamlit.app/

 ## Project Overview

This project focuses on analyzing the customer journey across the marketing funnel — from impressions to conversions — to understand:

Where users drop off
Which campaigns perform best
What factors impact conversion efficiency

The dashboard transforms raw campaign data into a decision-support system using storytelling and interactive visualizations.

## Types of Analysis Performed
**Funnel Analysis**: The funnel chart visualizes the user journey from impressions to conversions, helping identify where the highest drop-offs occur. It highlights critical stages where users disengage, enabling targeted optimization to improve overall conversion efficiency.

**Conversion Rate Analysis**: Conversion metrics evaluate how effectively users move through the funnel, from traffic to leads and from leads to customers. This helps assess the efficiency of both engagement and final conversion stages.

**Channel Performance Analysis**: Bar charts compare campaign performance across channels using conversion rate and ROI. This analysis helps identify top-performing channels for scaling and underperforming ones that require optimization.

**Segmentation Analysis**: Traffic-to-lead conversion is analyzed across segments such as channel, audience, and language. These visualizations reveal which segments drive better engagement and lead generation.

**Trend Analysis (Conversion Over Time)**: The line chart shows conversion trends over time, helping identify performance patterns, stability, and recent declines. It supports better decision-making by tracking campaign effectiveness over periods.

## How to Use the Dashboard
Use the interactive slider/filter to explore campaign performance based on conversion thresholds
Analyze the funnel chart to identify drop-off stages
Compare channels and segments to find high-performing areas
Use the trend analysis to monitor performance over time
Refer to the insights section for key findings and recommended actions

## Key Insights Derived
Significant user drop-off occurs at specific funnel stages, impacting overall conversion
Certain campaign channels consistently deliver higher conversion rates
Segment-based analysis reveals variation in engagement across audience and language
Conversion trends indicate performance stability with recent fluctuations
